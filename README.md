# My Modern Next.js Portfolio

A fully-featured, dynamic personal portfolio website built with **Next.js**, **MongoDB**, and **Tailwind CSS**.  
It includes a secure admin dashboard for managing projects, dynamic animations, and is optimized for deployment on **Vercel**.

This project was created to showcase my skills and projects in a modern, attractive, and interactive way.  
It goes beyond a static site by including a full **CRUD (Create, Read, Update, Delete)** system for projects, managed through a secure, password-protected admin panel.

## âœ¨ Features

- **Modern UI**: A beautiful, dark-themed, single-page layout with smooth scrolling and animations.  
- **Dynamic Animations**:
  - Introductory loading animation with a flipping airplane  
  - Scroll-reveal animations for each section  
  - Interactive 3D tilt effect on service cards  
  - Flipping animation on the "About Me" profile picture  
- **Project Management (CRUD)**:
  - Secure admin dashboard to add, view, edit, and delete projects  
  - Projects stored and managed in MongoDB  
- **Secure Admin Authentication**:
  - Admin login system using **JWT** with access/refresh tokens  
  - Credentials securely hashed and stored in MongoDB  
  - Protected routes using **Next.js Middleware**  
- **Cloud-Based Image Uploads**: Images uploaded to **Cloudinary** for optimization and global availability  
- **Fully Responsive**: Works seamlessly on mobile, tablet, and desktop  
- **SEO Optimized & Production Ready**: Built with Next.js for fast performance and easy deployment on Vercel
